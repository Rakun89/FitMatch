# FitMatch
FitMatch is a BMI-based exercise recommendation system that utilizes the K-Nearest Neighbor (KNN) algorithm to provide personalized workout suggestions based on users' physical conditions.

BMI-Based Exercise Recommendation System Using K-Nearest Neighbor

FitMatch is a web-based recommendation system designed to assist users in selecting appropriate exercise activities based on their Body Mass Index (BMI) and physical characteristics.

This system applies the K-Nearest Neighbor (KNN) classification method to analyze user data and generate personalized exercise recommendations that align with their physical condition.
